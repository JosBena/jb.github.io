Environmental Pong (Technologies Used: Unity, GitHub, RabbitMQ, Mirror, Weather API, PHP, SQL, Html)

Alongside a team (4 people) created an online multiplayer Game of Pong game that connected to a weather API to change both the environment and the weapons used. Furthermore, this game had a webpage where you could log in to check your score and buy cosmetic upgrades like hats for the game.

VR Pacman Horror Game (Technologies Used: Unity, GitHub, Blender)

Alongside a team (2 people) created an oculus VR Horror experience with rogue-lite elements that had procedurally generated mazed of different sizes. Furthermore, the game supported a system of collectible coins and meta points to obtain items, and powerups using the coins, and upgrade said items and powerups using the meta points after a run completion. According to

surveys of the game, it was scary and although could use some tweaking in some visual aspects it had potential.

Modifications on Quake 2 and Quake 4 (Technologies Used: QuakeC, C, C++)

Made 1 mod for Quake 2 (in C) and Quake 4 (in C++) each. The first has to do with replacing several weapons in the game with upgradable weapons, in addition to an upgrade system that improves the player attributes and weapons by using the souls of the enemies defeated. The Second has to do with a much simpler class creation, dealing with 3 different classes with 3 different abilities and weapons

per class.

Build A VR Forensic Application (Technologies Used: Unity, Unreal Engine 5, Github, GitLab, Point Clouds)

Created an Application on both UE 5 and Unity that deals with forensics to demonstrate how Forensics works to students, this was a project that used real-world scans converted to point

clouds to not only create better models but to build a more realistic scene. The project was later switched to be only on UE5 but I did most of the coding and built most of the assets to use on it.
